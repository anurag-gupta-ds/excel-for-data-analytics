# 🌫️ India Air Quality Analysis Dashboard

An interactive Excel dashboard analyzing real-time air quality data across India using official CPCB data from data.gov.in.

---

## 📌 Objective
Analyze pollutant levels and AQI categories across Indian cities and states to identify pollution patterns and hotspots.

---

## 📂 Dataset
- **Source:** Central Pollution Control Board (CPCB) via [data.gov.in](https://data.gov.in)
- **Records:** 3,163 entries
- **Coverage:** 36 states, 269 cities
- **Pollutants:** PM2.5, PM10, SO2, NO2, NH3, CO, OZONE

---

## 🛠️ Tools Used
- Microsoft Excel
- Pivot Tables
- Slicers
- Excel Charts (Bar, Pie, Line)

---

## 📊 Dashboard Features
- KPI Cards (Total Records, States, Cities, Highest AQI)
- Top 10 Most Polluted Cities
- Top 10 States by Average AQI
- AQI Category Distribution (Pie Chart)
- Average Pollutant Levels (Line Chart)
- Interactive Slicers (City, State, AQI Category)

---

## 🔍 Key Insights
- **Jaisalmer** is the most polluted city (AQI: 88.86)
- **82%** of cities fall under the "Good" AQI category
- **PM10** is the most dangerous pollutant (avg: 74.91)
- **Jharkhand** has the highest average state AQI
- **NH3** recorded the lowest average pollutant level (5.22)
