# 📊 Sales Executive Target Dashboard

An interactive Excel dashboard analyzing sales executive performance, targets, and achievements built while learning Excel from Satish Dhawale's complete Excel course.

---

## 📌 Objective
Track and analyze sales executive performance against targets across regions and time periods.

---

## 📂 Dataset
- **Type:** Sales executive performance data
- **Contains:** Sales targets, achievements, executive details

---

## 🛠️ Tools Used
- Microsoft Excel
- Pivot Tables + Pivot Charts
- Slicers
- Conditional Formatting
- Excel Formulas (IF, VLOOKUP, SUMIF, COUNTIF)
- Excel Dashboard Design

---

## 📊 Dashboard Features
- Sales vs Target comparison
- Executive wise performance tracking
- Region wise analysis
- Achievement % calculation
- Interactive Slicers for filtering
- KPI Cards

---

## 📚 What I Learned
- Data cleaning and preparation
- VLOOKUP and INDEX-MATCH
- Pivot Tables and Charts
- Dashboard design principles
- Slicer connectivity
- Conditional formatting for KPIs
