# 📊 Excel Projects

A collection of Excel dashboards and data analysis projects built as part of my Data Science and Data Analytics learning journey.

---

## 📁 Projects

### 1. 🌫️ India Air Quality Analysis Dashboard
- **Dataset:** Official CPCB data from data.gov.in
- **Tools:** Excel, Pivot Tables, Slicers, Charts
- **Highlights:** Analyzed AQI across 36 states, 269 cities with interactive slicers
- **Key Finding:** Jaisalmer most polluted city, PM10 most dangerous pollutant

### 2. 📈 Sales Executive Target Dashboard
- **Dataset:** Sales executive performance data
- **Tools:** Excel, Pivot Tables, VLOOKUP, Conditional Formatting
- **Highlights:** Sales vs target tracking with KPI cards and slicers
